# Samurai System

A modern web application built with Next.js, React, and Tailwind CSS, featuring smooth animations powered by GSAP and responsive design.

## 🚀 Features

- ⚡ Next.js 14 for server-side rendering and static site generation
- 🎨 Styled with Tailwind CSS for rapid UI development
- ✨ GSAP animations for smooth, performant transitions
- 🔄 React Intersection Observer for scroll-based animations
- 🎭 Swiper.js for touch-enabled sliders and carousels
- 🎯 Responsive design that works on all devices

## 🛠️ Tech Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS
- **Animation**: GSAP with React integration
- **UI Components**: Custom components with React Icons
- **Package Manager**: npm/yarn/pnpm/bun

## 📦 Prerequisites

- Node.js 16.8 or later
- npm (v7 or later), yarn, or pnpm
- Git

## 🚀 Getting Started

1. **Clone the repository**

   ```bash
   git clone https://github.com/yourusername/samurai-system.git
   cd samurai-system
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npm run dev
   ```

4. **Open [http://localhost:3000](http://localhost:3000)** in your browser to see the result.

## 🏗️ Project Structure

- `/app` - Application pages and routing
- `/app/_components` - Reusable React components
- `/public` - Static assets (images, fonts, etc.)
- `/node_modules` - Dependencies (automatically generated)
- `tailwind.config.js` - Tailwind CSS configuration
- `next.config.js` - Next.js configuration

## 🧪 Building for Production

```bash
npm run build

```
